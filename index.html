PROYECTOS

Digital Tech

Nov 2021- Mar 2022

Digital Tech es un proyecto llevado a cabo durante el curso de Desarrollador Web Full Stack de Digital House. Este proyecto Ecommerce demuestra mis habilidades en el diseño y desarrollo web, integración de API, manejo de bases de datos y gestión de usuarios. Para asegurar un desarrollo eficiente y estructurado, implementamos metodologías ágiles como Scrum. En el frontend, utilizamos HTML5 para la estructura, CSS3 para el estilo y JavaScript para la interactividad, asegurando una experiencia de usuario dinámica y fluida sin depender de frameworks como Bootstrap. En el backend, empleamos Node.js junto con Express.js para construir una API robusta y escalable que facilita la comunicación con la base de datos MySQL. Se aplicaron prácticas de seguridad con HTTPS y autenticación de usuarios para proteger la información confidencial.

Artemisa Hotel

May 2024 - Jun 2024

Artemisa Hotel es el proyecto que desarrollé junto a mi equipo en la materia de Introducción al Desarrollo de Software de mi carrera actual. Este proyecto fue diseñado para explorar y aplicar los principios del desarrollo web. La plataforma ofrece una experiencia interactiva donde los usuarios pueden explorar servicios, opciones de habitaciones y completar formularios personalizados. Utilizamos Flask como framework principal para la aplicación web. Implementamos una API RESTful en Flask para manejar la lógica del servidor y proporcionar datos dinámicos a la interfaz de usuario. En cuanto al frontend, nos enfocamos en mejorar la experiencia del usuario con una interfaz intuitiva y atractiva. Además, utilizamos MySQL para almacenar y gestionar eficientemente la información de los usuarios, reservaciones y otros datos relevantes del hotel. ste proyecto no solo nos permitió aplicar los conocimientos teóricos adquiridos en clase, sino también mejorar nuestras habilidades en el desarrollo de software colaborativo, el diseño de interfaces de usuario y la integración de diversas tecnologías para crear una aplicación web completa y funcional.

Tesoro Binario

Sep 2023 - Oct 2023

Tesoro Binario es un proyecto que desarrollé indivisualmente en la materia Algoritmos y Programacion II. Este proyecto simula un juego de estrategia para dos jugadores en un tablero de 20x20 casillas. Cada jugador coloca y oculta cuatro tesoros mientras intenta descubrir y mover tesoros del oponente usando espías. La interfaz de usuario, totalmente textual, permite la visualización y exportación del estado del tablero después de cada turno. Implementado en Python con manejo avanzado de archivos de texto para el estado del tablero y lógica de juego basada en algoritmos y estructuras de datos. Desarrollé mis habilidades en programación orientada a objetos, algoritmos y gestión de estado en aplicaciones interactivas.

Pasapalabra

May 2023- Jul 2023

Pasapalabra es un proyecto que desarrollé con mi equipo en la materia Algoritmos y Programación I. Un proyecto realizado en Python que permite que hasta cuatro jugadores puedan competir por turnos para adivinar palabras asociadas con diferentes letras del alfabeto, cada una con su definición correspondiente. Calcula puntajes parciales y totales de cada jugador al finalizar cada ronda. Se ofrece la opción de jugar múltiples rondas si los jugadores desean continuar. Incluye la capacidad de configurar valores predeterminados. Utiliza la consola para mostrar tableros de juego, resultados de cada turno y puntajes finales. El proyecto me permitió aprender y aplicar conceptos de trabajo en equipo, gestión de archivos, manejo de estructuras de datos complejas como diccionarios y listas, y desarrollo de lógica de juego interactiva.

Mercado Liebre
